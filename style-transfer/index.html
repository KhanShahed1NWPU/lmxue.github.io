Abstract: In this paper, we present a cross-speaker style transfer end-to-end speech synthesis model using cycle consistent netrowk. Built upon an end-to-end multi speaker model where Variational Autoencoder (VAE) is employed to learn the latent representation accounting for the speaking style of the audio, we firstly augment the model with an unpaired audio input and an unpaired transferred output, so as to make the cross-speaker style transfer behavior seen during model training. Secondly, we propose a cycle consistent network to solve the problem that the transferred output is hard to train due to lack of effective target to constrain. Experimental results shows that the proposed cycle model can achieve much better performance than Global Style Token (GST) and VAE based model in cross-speaker style transfer both for parallel transfer and unparallel transfer.
